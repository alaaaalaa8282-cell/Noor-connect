# 🌙 Noor Connect - Your Ultimate Islamic Companion

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT) [![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/darkmaster0345/Noor-connect)

**Noor Connect** is a comprehensive, beautiful, and privacy-focused Islamic application designed to help you stay connected with your faith. Built with modern web technologies and optimized for mobile experiences via Capacitor, it serves as your digital companion for daily spiritual growth.

---

## ✨ Key Features

### 📖 Holy Quran
Explore the Word of Allah with a beautiful, readable interface. Includes detailed Surah information and recitations.

### ⏲️ Accurate Prayer Times & Adhan
Localization-based prayer timings with customizable Adhan notifications and a global prayer alarm system.

### 🕋 Qibla Finder
A high-precision compass to help you find the Kaaba's direction (Qibla) from anywhere in the world.

### 📻 Quran Radio & Live Streams
Listen to 24/7 Quran recitations from global stations or watch live streams from Makkah and Madinah.

### 🌙 Ramadan Mode
Specialized tools for the blessed month, including Suhoor/Iftar timings and spiritual goals.

### 📿 Spiritual Tools
- **Digital Tasbeeh**: An interactive counter for your daily Dhikr.
- **Qaza Tracker**: Easily log and track missed prayers.
- **Habit Tracker**: Build and maintain consistent spiritual habits.
- **Prayer Stats**: Visualize your consistency and spiritual progress.

### 📚 Knowledge & Wisdom
- **Duas & Hadith**: A rich collection of authentic supplications and prophetic traditions.
- **99 Names of Allah**: Reflect on the beautiful attributes of Allah (SWT).
- **Islamic Quiz**: Test and expand your knowledge of Islam.
- **E-Books**: Access a library of Islamic literature on the go.

### 💰 Zakat Calculator
Quickly and accurately calculate your Zakat and wealth distribution.

## 🚀 Technical Excellence

This application is built using a modern, high-performance stack:

- **Frontend**: [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Mobile Foundation**: [Capacitor](https://capacitorjs.com/) (Android)
- **Reliability**: PWA support with Service Workers for offline capabilities.
- **Calculations**: Powered by the [Adhan](https://github.com/batoulapps/adhan-js) library for precise timing.

## 🛠️ Getting Started

### Prerequisites
- **Node.js**: (v18+)
- **npm** or **bun**
- **Android Studio**: For building the native Android APK.

### Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/darkmaster0345/Noor-connect.git
   cd Noor-connect
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npm run dev
   ```

### Building for Mobile (Android)

1. **Prepare the web build:**
   ```bash
   npm run build
   ```

2. **Sync with Capacitor:**
   ```bash
   npx cap sync android
   ```

3. **Open in Android Studio:**
   ```bash
   npx cap open android
   ```
